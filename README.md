# 💳 Payment Processing System

A complete payment processing system built with Python Flask and Stripe API.

## 🚀 Features

- 🔐 **User Authentication** - JWT-based login and registration
- 💰 **Stripe Integration** - Secure payment processing
- 📊 **Payment Dashboard** - View transaction history
- 🗄️ **Database Storage** - SQLite with User and Payment models
- 🎨 **Modern UI** - Clean, responsive frontend

## 🛠️ Tech Stack

- **Backend:** Python, Flask
- **Payment:** Stripe API
- **Database:** SQLite, SQLAlchemy
- **Authentication:** JWT, bcrypt
- **Frontend:** HTML, CSS, JavaScript

## 📋 Prerequisites

- Python 3.8+
- Stripe Account (free)

└── requirements.txt       # Python dependencies
